# Count Vowels

## Problem
Write a function that counts the number of vowels in a given string. Vowels are 'a', 'e', 'i', 'o', and 'u' (both lowercase and uppercase).

## Examples
**Example 1:**
- Input: "hello"
- Output: 2 (e, o)

**Example 2:**
- Input: "Beautiful"
- Output: 5 (e, a, u, i, u)

**Example 3:**
- Input: "Sky"
- Output: 0 (no vowels)
